DOCUMENT: TRI-S-AUDIT GLOSSARY (EN · RU · UZ-Latin) — ISA/IFRS CORE (B4-LEVEL)

Purpose
-------
Unify terminology across our tri-lingual environment so every deliverable (audits, TCWG, legal matrix, KAM, PBC) reads consistently and audit-grade.  
Primary language of standards: **English**. Russian/Uzbek are controlled translations aligned to IFRS/ISA usage (Uzbek in **Latin** per 02_StyleGuide).

Usage Rules
-----------
- **Do not invent terms.** If no stable RU/UZ analogue exists, keep EN + short note.
- Prefer IFRS/ISA official wording in RU (e.g., IAS 37: “оценочное обязательство”, not “резерв”).
- Uzbek: keep neutral technical calques; avoid colloquialisms; Latin only.
- Capitalization: sentence case (except standard names). Abbrev at first use: “Those charged with governance (TCWG)”.
- Numbers/units per 02_StyleGuide; cross-link to 05/06/07/11/12 when drafting.

File Schema (for CSV/XLSX exports)
----------------------------------
| Term_EN | Term_RU | Term_UZ | Notes_Usage | Ref |
Acceptance tests: UTF-8; one term/row; no merged cells; EN unique; Ref uses standard names (ISA/IAS/IFRS).

FALSE FRIENDS & STYLE GUARDRAILS
--------------------------------
- **Provision (IAS 37)** = “оценочное обязательство” (RU) / “baholanadigan majburiyat” (UZ).  
  **Do not** use “резерв” as the primary term (allowed only in lay prose with note).
- **Allowance for doubtful debts / ECL**: prefer “резерв по ожидаемым кредитным убыткам (ECL)”; avoid legacy “сомнительные долги” unless explaining.
- **Recognition vs Disclosure vs Presentation** must not be conflated: признание / раскрытие / представление; tan olish / oshkor etish / taqdim etish.
- **Write-down vs Write-off**: уценка (write-down) vs списание (write-off); kamaytirish vs hisobdan chiqarish.
